# 行为准则（Contributor Covenant v2.1）

## 我们的承诺

我们承诺为所有人提供无骚扰的参与环境，无论年龄、体型、可见或隐性残疾、族群、性别、性取向、经验水平、教育、社会与经济地位。

## 我们的准则

**鼓励的行为：**
- 使用友好、包容的语言
- 尊重不同的观点与经验
- 优雅地接受建设性批评
- 以社区整体利益为重

**不可接受的行为：**
- 骚扰、歧视或侮辱性言论
- 公开或私下的骚扰
- 未经许可发布他人私密信息（如凭据、地址）
- 其他在专业场合中显然不当的行为

## 执行

维护者负责解释与执行本准则。涉及违规可联系维护者（见 `README.md`）。维护者有权删除、编辑或拒绝不符合本准则的内容与贡献。

## 署名

本准则基于 [Contributor Covenant](https://www.contributor-covenant.org)，版本 2.1。
